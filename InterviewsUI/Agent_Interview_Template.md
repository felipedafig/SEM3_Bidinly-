### Scenario 1: Account Creation and First Impression

**Task:** "You want to list properties on Bidinly. Create an Agent account."

**Steps:**
1. Open the homepage
2. Navigate to sign up
3. Sign up as an Agent
4. Log in
5. Explore the agent dashboard

**Observe:**
- [ ] Is the Agent role clearly explained?
- [ ] Is the sign-up flow smooth?
- [ ] Are expectations after login clear?
- [ ] Do they understand what an Agent can do?
- [ ] Is the dashboard helpful?

**Follow-up Questions:**
- Was it clear what an Agent can do on Bidinly?
- Did anything slow you down during registration?
- What was your first impression of the agent dashboard?
- Did you understand the difference between Buyer and Agent accounts?

### Scenario 2: Adding and Managing Properties

**Task:** "Add a new property listing and manage your existing properties."

**Steps:**
1. Navigate to your properties page
2. Add a new property with all required details
3. Use filters to find a specific property
4. View property details
5. Review bids received for a property

**Observe:**
- [ ] Is the "Add Property" action easy to find?
- [ ] Are required fields understandable?
- [ ] Is the property creation form intuitive?
- [ ] Can they effectively filter their properties?
- [ ] Is property management clear?
- [ ] Can they easily view bids for their properties?

**Follow-up Questions:**
- How confident did you feel adding a property?
- Were any fields unclear or unnecessary?
- What tools would help you manage properties better?
- Was the property management interface clear?
- How easy was it to find and review bids on your properties?
